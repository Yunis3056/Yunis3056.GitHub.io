Java基础知识

java 概述

什么是 java

Java 是 Sun Microsystems 于1995年首次发布的一种编程语言和计算平台。编程语言还比较好理解，那么什么是计算平台呢？

计算平台是在电脑中运行应用程序（软件）的环境，包括硬件环境和软件环境。一般系统平台
包括一台电脑的硬件体系结构、操作系统、运行时库。

Jva是快速，安全和可靠的。从笔记本电脑到数据中心，从游戏机到科学超级计算机，从手机到互联
网，Java无处不在！Java主要分为三个版本

  • JavaSE(J2SE)Uava2 Platform Standard Edition,java平台标准版）
  • JavaEE(J:2EE)Java2 Platform,Enterprise Edition,java平台企业版）
  • JavaME(J2 ME)(Java2 Platform Micro Edition,java平台微型版)。

    事情？
    起床、穿衣服、洗脸刷牙，吃饭，去学校。一般是顺序性的完成一系列动作。

      这是特征二。

    • Java语言具有功能强大和简单易用两个特征，现在企业级开发，快速敏捷开发，尤其是各种框架
      的出现，使Java成为越来越火的一门语言。这是特点三。

    从设计的角度上来说，所有的语言都是设计用来把人类可读的代码转换为机器指令。动态语言是为了能

  3. Java 能够容易实现多线程

    Java是一门高级语言，高级语言会对用户屏蔽很多底层实现细节。比如Java是如何实现多线程的。从

    而我认为Java应该是在用户空间实现的多线程，内核是感知不到Java存在多线程机制的。这是特
    点六。

  4. Java 具有高性能

    我们编写的代码，经过javac编译器编译称为字节码(bytecode),经过JVM内嵌的解释器将字节码
    转换为机器代码，这是解释执行，这种转换过程效率较低。但是部分JVM的实现比如Hotspot JVM
    都提供了JIT(Just-In-Time)编译器，也就是通常所说的动态编译器，JIT能够在运行时将热点代
    码编译机器码，这种方式运行效率比较高，这是编译执行。所以Jva不仅仅只是一种解释执行的语
    言。这是特点七。

  5. Java 语言具有健壮性

    Java的强类型机制、异常处理、垃圾的自动收集等是Java程序健壮性的重要保证。这也是Java与C
    语言的重要区别。这是特点八。

  6. Java 很容易开发分布式项目

    Java语言支持Internet应用的开发，Java中有net api,它提供了用于网络应用编程的类库，包括
    URL、URLConnection、Socket、ServerSocket等。Java的RMI（远程方法激活）机制也是开发分布
    式应用的重要手段。这是特点九。

java 开发环境

JDK

JDK(Java Development Kit)称为Java开发包或Java开发工具，是一个编写Java的Applet
小程序和应用程序的程序开发环境。JDK是整个Java的核心，包括了Java运行环境（Java Runtime
Environment),一些Java工具和Java的核心类库(Java API)

Oracle提供了两种Java平台的实现，一种是我们上面说的JDK,Java开发标准工具包，一种是
JRE,叫做Java Runtime Environment,.Java运行时环境。JDK的功能要比JRE全很多。

JRE

JRE是个运行环境，JDK是个开发环境。因此写Java程序的时候需要JDK,而运行Java程序的时候就需要JRE。而JDK里面已经包含了JRE,因此只要安装了JDK,就可以编辑Jva程序，也可以正常运行Java程序。但由于JDK包含了许多与行无关的内容，占用的空间较大，因此运行普通的Java程序无须安装JDK,而只需要安装JRE即可。

标识符的命名规范

标识符的命名规范：

  • 类名或接口名：首字母大写 （大驼峰式）
  • 变量或方法名：由一个单词组成全部小写，由多个单词组成，从第二个单词开始首字母大写。（小驼峰式）
  • 包名：
    • 单级包：全部小写
    • 多级包：全部小写中间用.连接 一般用公司域名的反写。
  • 常量名：每个单词的每个字母都要大写，多个单词只用用_连接。_
  • 标识符：26个大写字母 26个小写字母 0-9数字 $,_。 见名知意。

数据类型

Java有 四类八种数据类型：

  • 整数型

    byte：用于表示8位有符号整数，取值范围为-128到127，默认值为0。

    short：用于表示16位有符号整数，取值范围为-32768到32767，默认值为0。

    int：用于表示32位有符号整数，取值范围为-2147483648到2147483647，默认值为0。

    long：用于表示64位有符号整数，取值范围为-9223372036854775808到9223372036854775807，默认值为0L。

  • 浮点型

    float：用于表示单精度浮点数，取值范围为3.4e-38到3.4e+38，默认值是 0.0f。

    double：用于表示双精度浮点数，取值范围为1.7e-308到1.7e+308，默认值是 0.0d。

  • 字符型

    char：用于表示16位Unicode字符，取值范围为’\u0000’到’\uffff’。

  • 布尔型

    boolean：用于表示布尔类型，取值范围为true和false。

运算符和表达式

Java有各种各样的运算符，包括算术运算符、关系运算符、逻辑运算符、位运算符等等。表达式由操作数和运算符组成。

  1. 算术运算符：用来进行数值计算的运算符，包括加(+), 减(-), 乘(*), 除(/), 取余(%), 自增(++), 自减(–), 等等。
  2. 关系运算符：用来比较两个值之间的关系，包括等于(==), 不等于(!=), 大于(>), 小于(<), 大于等于(>=), 小于等于(<=)等。
  3. 逻辑运算符：用来进行布尔逻辑运算的运算符，包括逻辑与(&&), 逻辑或(||), 逻辑非(!)等。
  4. 位运算符：用来操作二进制数的运算符，包括按位与(&), 按位或(|), 按位异或(^), 取反(~), 左移位(<<), 右移位(>>), 无符号右移位(>>>)等。
  5. 赋值运算符：用来给变量赋值的运算符，包括简单赋值(=), 加等于(+=), 减等于(-=), 乘等于(*=), 除等于(/=), 取余等于(%=)等。
  6. 条件运算符：用来进行条件判断的运算符，也称三元运算符，格式为(condition ? value1 : value2)，如果条件成立，则返回value1，否则返回value2。
  7. instanceof运算符：用来判断对象是否属于某个类或其子类的运算符，格式为(object instanceof class)，如果是则返回true，否则返回false。

2、循环语句

循环语句用来重复执行指定的代码块，包括for语句、while语句和do-while语句。

  • for语句可以指定循环的起始条件、终止条件和每次循环的迭代操作。
  • while语句在每次循环开始前，先判断条件是否成立，如果成立则执行代码块，否则跳出循环。
  • do-while语句和while语句类似，但是先执行一次代码块，再判断条件是否成立。
  • 三种循环的区别：
  1. for循环和while循环如果条件不成立一次循环语句体都不执行。do while循环即使条件不成立，也会执行一次循环语句体。

2.在已知循环次数的时候使用推荐使用for，循环次数未知的时推荐使用while。

3、跳转语句

跳转语句用来跳转到程序的其他位置执行代码，包括break语句、continue语句和return语句。

break语句用于跳出当前循环或switch语句。

位置：用在switch语句中或者循环中

作用：彻底跳出循环。

注意：break下面不允许有其它语句出现，因为永远执行不到。

continue语句用于跳过当前循环的剩余部分，直接进入下一次循环。

位置：循环中

作用：跳过当前循环，执行下一次循环。

注意：continue下面不允许其它语句出现，因为永远执行不到。

return语句用于退出方法，并返回一个指定的值。

位置：方法中，循环中。

作用：结束方法。

注意：return下面不允许有其它语句出现，因为永远执行不到。

数组

Java中的数组是一种数据结构，可以用来存储同一类型的数据元素。它们在内存中以连续的方式存储，并且可以通过下标进行访问和修改。

Java数组有以下特点：

  • 长度固定

    Java数组的长度在创建时就已经确定，并且不能改变。因此，在创建数组时，必须指定它的长度。

  • 类型统一

    Java数组只能存储同一种数据类型的元素，不允许混合不同类型的元素。

  • 从0开始编号

    Java数组的下标从0开始，最大下标为长度减1。这意味着数组中的第一个元素的下标是0，第二个元素的下标是1，以此类推。
